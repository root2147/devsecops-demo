# DevSecOps Demo

This repository is an academic DevSecOps demonstration for a Level 6 cybersecurity examination.

The goal is to demonstrate "shifting security left" by using GitHub Actions to automatically scan code for security issues before deployment.

## What This Demo Shows

- A deliberately vulnerable Python application
- Automated security scanning in a CI/CD pipeline
- SAST scanning for insecure source code
- SCA scanning for vulnerable dependencies
- A security gate that blocks unsafe code
- A remediation cycle where vulnerabilities are fixed and the pipeline passes

## Important Safety Note

This project is for education only.

Any passwords, API keys, or vulnerabilities used in this repository are fake and intentionally created for demonstration purposes.
Never commit real secrets to GitHub.
